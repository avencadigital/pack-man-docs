## Why Use Extensions?

Extensions bring Pack-Man's powerful dependency analysis directly into the tools you use every day:

- **Seamless Integration** - No need to switch contexts or copy/paste files
- **Real-time Analysis** - Get instant feedback as you browse or code
- **Enhanced Productivity** - Identify outdated dependencies without leaving your workflow
- **Visual Indicators** - See package status at a glance with color-coded badges

## Coming Soon

We're actively working on additional integrations:

- **CLI Tool** - Command-line interface for CI/CD pipelines
- **GitHub Action** - Automated dependency checks in your workflows
